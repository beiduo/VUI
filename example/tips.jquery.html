调用方法：

 

html：添加属性：

j_tips="title:标题|content:内容|followMouse|width:宽度|fontSize:字号|placement:显示方向"

属性之间用|隔开，无先后顺序。"content:内容"中，前面的"content:"可以省略

只有content是必须的，所有最简单的调用方式是 j_tips="提示内容"即可

 

JS调用

 

VUI.tips(elem, option);

 

option用做高级自定义，如果不需要，通常省略。

 

elem可以是一个dom对象，也可以是一个dom对象的数组，也可以是CSS选择器（类似jquery）。会根据传入参数的类型自动进行判断

 

例如本演示中的例子是：

VUI.tips('#tips p');

 

附：高级自定义

 

通过option参数可以对tip提示进行更深的自定义，所有VUI.tips.defaults对象的属性和方法都可以在这里进行重写

 

优先级： j_tips属性 > option参数 > VUI.tips.defaults

 

例如

 

VUI.tips(elem, {

    skin: '皮肤', //为提示框加上vui-tips-skin的classname，方便定义不同样式

    tpl: function(){

        var _tpl = '<em>.</em><div class="vui-tips-cntr"></div>'; //例如需要箭头指示，可以这样自定义模板，再建立css样式

        return _tpl;

    }

    //你还可以照这样重新定义show\hide方法以便加上动画过渡，或者重新定义setContent方法添加按钮，或者重新定义events方法修改相关事件处理机制，whatever you want.

});
